== Description ==

This plugin will allow the administrator to login as different user

== Installation ==

1. Upload `admin-login-as-different-user` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It can be found under Tools->Login As.....
